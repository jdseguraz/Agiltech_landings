# GovWapp Assets

Carpeta de recursos para la sublanding gov-tech

## Estructura:

- **images/**: Imágenes, fotos, screenshots
- **icons/**: Íconos personalizados, logos en diferentes formatos

## Formatos recomendados:

- Logo: SVG, PNG (transparente)
- Imágenes: WebP, PNG, JPG
- Íconos: SVG preferiblemente
